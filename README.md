### git 提交步骤

1. git add . 用来将你所改动的代码 提交到本地 待提交仓库
2. git commit -m"\*\*\*\*" 用来将本来的待提交仓库 提交到本地提交仓库 m 后面跟的是提交描述
3. git push 将 commit 提交到线上
